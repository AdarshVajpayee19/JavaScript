An Event is signal that something has happened all the DOM nodes generate such signals.

Some Important DOM Events are:

    Mouse Events:

  • Click
  • Contextmenu(Right Click)
  • mouseover/mouseout
  • mousedown/mouseup
  • mousemove

    KeyBoard Events:

  • KeyUp
  • Keydown

    Form Events:

  • Submit
  • Focus etc...

    Document Events:

  • DOMContentLoaded
