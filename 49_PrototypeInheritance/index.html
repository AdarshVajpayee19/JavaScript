Prototype Inheritance

JavaScript objects have special property called prototype that is either null or references another object.

When we try to read a property from a prototype and its missing,JavaScript automatically takes it from the prototype.This is Called as "Prototype Inheritance."

setting Prototype

We can Set Prototype by setting __proto__ Now if we read property from the object which is not in object and is present in the prototype,JavaScript will take it from prototype.


If we have a method in object, it is called from the object.If its missing in object and present in prototype, its is called from prototype.
